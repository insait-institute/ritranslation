Δεν είμαι φυσικός ομιλητής της ελληνικής γλώσσας, πώς θα έλεγες αυτό το κείμενο στα <target_language>:
{
    "Αρχικό_Κείμενο": """<original_text>"""
}
Με ενδιαφέρει πώς εσύ, ως φυσικός ομιλητής, θα το έλεγες στη μητρική σου γλώσσα, καθώς και πώς θα διατηρούσες το νόημα του κειμένου. Θα ήθελα να δω πώς μπορεί να μεταφραστεί αυτό το κείμενο έτσι ώστε να ακούγεται φυσικό στα ελληνικά, και φυσικά να είναι σωστή η γραμματική (π.χ. πτώσεις και γένη). Τώρα εκτύπωσε τα μεταφρασμένα δεδομένα στα <target_language> στην ακόλουθη μορφή και, παρακαλώ, εμφάνισε μόνο το κείμενο μεταφρασμένο στα ελληνικά:
Κείμενο: μεταφρασμένο_κείμενο_σε_<target_language>
