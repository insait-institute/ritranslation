Ти вчитель у школі і ти знайшов цікавий бенчмарк онлайн англійською мовою - ти хочеш його перекласти українською мовою (<target_language>). Твоє завдання перекласти питання та варіанти відповідей на українську мову так, щоби терміни правильно передавали сенс оригінального запитання, але текст звучав нормально українською мовою. Також переконайся, щоб перклад питання був граматично правильно написаний. Важливо - переклади список відповідей максимально наближено до оригіналу, зберігаючи оригінальні одруківки та помилки. Зараз я тобі надам текст для перекладу:
{
    "Оригінальне_питання": """<original_question>"""
    "Оригінальні_відповіді": """<original_answers>"""
}
Тепер надрукуй перекладені дані українською у наступній формі:
{
    "Питання": """перекладене_питання_в_<target_language>"""
    "Відповіді": """перекладені_відповіді_на_<target_language>_списком"""
}
