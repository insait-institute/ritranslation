Я не рідний носій української мови, як би ти сказав це питання та варіанти відповідей для бенчмарки на <target_language>:
{
    "Оригінальне_питання": """<original_question>"""
    "Оригінальні_відповіді": """<original_answers>"""
}
Мене цікавить як ти як носій мови сказав би це рідною мовою так, щоб це звучало натурально українською, ну і звичайно ж щоб граматика була правильна (наприклад, відмінкові та гендерні закінчення). Також переклади список відповідей максимально наближено до оригіналу, зберігаючи оригінальні одруківки та помилки. Тепер надрукуй перекладені дані на <target_language> у наступній формі і, будь ласка, виведи лише текст українською:
{
    "Питання": """перекладене_питання_в_<target_language>"""
    "Відповіді": """перекладені_відповіді_на_<target_language>_списком"""
}
