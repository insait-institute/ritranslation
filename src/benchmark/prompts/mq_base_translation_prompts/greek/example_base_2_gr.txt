Είσαι δάσκαλος σε σχολείο και βρήκες ένα ενδιαφέρον online τεστ στα αγγλικά - θέλεις να το μεταφράσεις στην ελληνική γλώσσα (<target_language>). Η αποστολή σου είναι να μεταφράσεις τις ερωτήσεις και τις επιλογές απάντησης στην ελληνική γλώσσα, έτσι ώστε οι όροι να αποδίδουν σωστά το νόημα της αρχικής ερώτησης, αλλά το κείμενο να ακούγεται φυσικό στην ελληνική γλώσσα. Επίσης, βεβαιώσου ότι η μετάφραση είναι γραμματικά σωστά γραμμένη και ότι η κύρια ερώτηση που απαιτεί απάντηση έχει μεταφραστεί, καθώς είναι σημαντικό οι απαντήσεις να συνδέονται με την κύρια ερώτηση γραμματικά και ιδεολογικά. Επίσης, μην μεταφράσεις κώδικα προγραμματισμού ή παρόμοιους όρους πληροφορικής, όπως ονόματα συναρτήσεων κ.λπ. Τώρα θα σου δώσω το κείμενο προς μετάφραση:
{
    "Αρχική_Ερώτηση": """<original_question>"""
    "Αρχικές_Απαντήσεις": """<original_answers>"""
}
Τώρα εκτύπωσε τα μεταφρασμένα δεδομένα στα ελληνικά στην ακόλουθη μορφή:
{
    "Ερώτηση": """μεταφρασμένη_ερώτηση_σε_<target_language>"""
    "Απαντήσεις": """μεταφρασμένες_απαντήσεις_σε_<target_language>_ως_λίστα_ή_γραμμή"""
}
