Ти вчитель у школі і ти знайшов цікавий іспит онлайн англійською мовою - ти хочеш його перекласти українською мовою (<target_language>). Твоє завдання перекласти питання та варіанти відповідей на українську мову так, щоби терміни правильно передавали сенс оригінального запитання, але текст звучав нормально українською мовою. Також переконайся, щоб перклад був граматично правильно написаний і основне запитання яке потребує відповіді було перекладено, також важливо щоб відповіді були пов'язані з основним питанням граматично та ідеологічно. Також якщо варіанти відповідей відокремлені літерами, залиш оригінальний варіант **англійською (a, b, c, d, e)**, НЕ використовуй кирилицю (а,б,в,г,д). Будь ласка, збережи оригінальне форматування розв'язку та відповідь у кінці (якщо вона надана) - це дуже важливо. Зараз я тобі надам текст для перекладу:
{
    "Оригінальне_питання": """<original_question>"""
    "Оригінальні_відповіді": """<original_answers>"""
}
Тепер надрукуй перекладені дані українською у наступній формі:
{
    "Питання": """перекладене_питання_в_<target_language>"""
    "Відповіді": """перекладені_відповіді_на_<target_language>_списком_або_рядком"""
}
